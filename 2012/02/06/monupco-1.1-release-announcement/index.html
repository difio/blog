Monupco 1.1 Release Announcement

Posted by Monupco Team @ Feb 6th, 2012

Today Monupco is pleased to announce the general availability of our updates monitoring service. Monupco provides user friendly interface to monitor available updates for your cloud applications and information about different package versions.

At the time of launch Monupco supports registration of Python applications hosted on Red Hat’s OpenShift Express platform. Client side tools and installation information can be found at our GitHub page.

Monupco itself is developed and hosted in the cloud and we’re working to support additional cloud vendors and programming languages so stay tuned for more news. We’re encouraging all early adopters to send us their feedback and feature requests via Twitter or GitHub.

While the service is operating normally it has limited support for cloud vendors and package types and is considered an alpha release. Monupco will remain free of charge during its Alpha and Beta periods.

Registration and login into our management console is straight forward using credentials from several popular social networking providers.
